# AI-Based Crop Recommendation for Farmers (SDG-2)

## 📌 Overview
This project implements an **AI-Based Crop Recommendation System** aimed at supporting **Sustainable Development Goal 2 (Zero Hunger)** by enabling data-driven and climate-resilient agricultural decision-making. The system recommends suitable crops based on **soil nutrients and climatic parameters**, helping farmers improve productivity and reduce crop failure risks.

To enhance real-world applicability and research relevance, the system integrates **climate-adaptive weather inputs**, **yield prediction**, and **Explainable AI (XAI)** to ensure transparency and robustness of recommendations under climate variability.

---

## 🎯 Objectives
- Recommend suitable crops based on soil and climate conditions  
- Improve agricultural productivity and sustainability  
- Adapt recommendations using real-time climate data  
- Provide explainable insights into AI-driven decisions  
- Support SDG-2: Zero Hunger through sustainable farming practices  

---

## 🧠 System Features
- Machine learning-based crop recommendation  
- Climate-adaptive input using weather data  
- Crop yield prediction module  
- Explainable AI for transparent decision-making  
- Comparative model evaluation and analysis

## 📊 Dataset
The system uses publicly available agricultural datasets containing:
- Soil parameters (N, P, K, pH)  
- Climatic features (temperature, rainfall, humidity)  
- Historical crop data

## 📈 Methodology
1. Data preprocessing and feature engineering  
2. Training multiple machine learning models  
3. Climate-adaptive input integration  
4. Explainable AI analysis  
5. Comparative and ablation-based evaluation
